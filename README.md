# IE10-support-audio-type

IE10 支持的音频格式如下。

| 格式 | IE 10 | IE 11 | Firefox 75.0 | Chrome 81.0 |
| ---- | ----- | ----- | ------------ | ----------- |
| mp3  | ✅    | ✅    | ✅           | ✅          |
| m4a  | ✅    | ✅    | ✅           | ✅          |
| aac  |       |       | ✅           | ✅          |
| wav  |       |       | ✅           | ✅          |
| flac |       |       | ✅           | ✅          |
| ogg  |       |       | ✅           | ✅          |
| wma  |       |       |              | ✅          |
